Obstacle Avoidance Robot

Project Summary

This is a basic robotics project where I built a mobile robot that avoids obstacles using two ultrasonic sensors. It's a learning exercise to practice embedded systems, sensor use, and simple programming for navigation. The robot uses the sensors to check space on the left and right, helping it turn based on available clearance.

üéØ Main Goals

The project focuses on using dual sensors to provide awareness of obstacles on both sides, enabling the robot to navigate in cluttered areas. This serves as a practical exercise in beginner robotics, with potential applications in areas like farming or simple automation.

üîß How It Works

  • Sensors: Two HC-SR04 sensors on the front corners, angled for wide coverage. They measure distances by timing sound echoes.
  • Decision Making: The Arduino code compares distances:
    • If blocked on both sides, turn to the clearer side.
    • If blocked on one side, turn away.
    • Stop if too close (under 20 cm), slow down if nearby (30-40 cm), go full speed if clear.
  • Motors: Two DC motors with an L298N driver for moving forward, turning, or stopping. Speed adjusts based on distance.
  • Extras: Simple filters to ignore sensor noise, handles stuck situations by turning around, runs in a loop.

üèóÔ∏è Hardware

  • Controller: Arduino Uno.
  • Sensors: 2x HC-SR04.
  • Driver: L298N.
  • Motors: 2x DC motors.
  • Base: Wheeled chassis with separate power for electronics and motors.
